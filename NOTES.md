Project Notes

Setup
Set your Jupyter kernel to the project root.
All file paths are relative to root, not the `notebooks/` folder.

Run Order
1. `data_acquisition.ipynb` — fetch S&P 500 price data & generate pairs
2. `screening.ipynb` — cointegration screening across 17,955 pairs
3. `dynamic_hedge.ipynb` — Kalman filter hedge ratios & dynamic spreads
4. `regime_detection.ipynb` — HMM market regime detection
5. `trading_signals.ipynb` — regime-conditioned entry/exit signals
6. `backtest_engine.ipynb` — full strategy simulation with realistic costs

Data
Large CSV files are not included in this repo due to size.
The only data file included is `data/sp500_top190_tickers.csv`.
All other data files are generated by running the notebooks in order.

Results Disclaimer
Reported metrics (Sharpe: 2.11, Return: 534%) are likely optimistic due to
look-ahead bias in the pipeline. This is my first end-to-end quant project.
The goal was building and understanding the full pipeline, not production-ready
results. Clean walk-forward validation is the next step.
